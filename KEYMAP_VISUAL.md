## Layer 3: ADJUST (RGB & Advanced Settings)

**Automatically activates when holding LOWER + RAISE together**

```
╭─────────┬─────┬─────┬─────┬─────┬─────╮                          ╭─────┬─────┬─────┬─────┬─────┬─────╮
│ BT CLR  │ BT1 │ BT2 │ BT3 │ BT4 │ BT5 │                          │     │     │     │     │     │     │
├─────────┼─────┼─────┼─────┼─────┼─────┤                          ├─────┼─────┼─────┼─────┼─────┼─────┤
│EXT PWR  │HUE -│HUE +│SAT -│SAT +│ EFF │                          │     │     │     │     │     │     │
├─────────┼─────┼─────┼─────┼─────┼─────┤                          ├─────┼─────┼─────┼─────┼─────┼─────┤
│         │BRI -│BRI +│     │     │     │                          │     │     │     │     │     │     │
├─────────┼─────┼─────┼─────┼─────┼─────┼────────╮      ╭─────────┼─────┼─────┼─────┼─────┼─────┼─────┤
│         │     │     │     │     │     │RGB TOG │      │         │     │     │     │     │     │     │
╰─────────┴─────┴─────┼─────┼─────┼─────┼────────┤      ├─────────┼─────┼─────┼─────┼─────┴─────┴─────╯
                       │     │     │     │        │      │         │     │     │     │
                       │     │     │     ├────────┤      ├─────────┤     │     │     │
                       │     │     │     │        │      │         │     │     │     │
                       ╰─────┴─────┴─────┴────────╯      ╰─────────┴─────┴─────┴─────╯

Encoders:  RGB Toggle                                               (not set)
```

**Key Features:**

- **Bluetooth:** Same as RAISE layer
- **RGB Controls:**
  - HUE -/+: Change color
  - SAT -/+: Change saturation (color intensity)
  - BRI -/+: Change brightness
  - EFF: Cycle through effects
  - RGB TOG: Turn underglow on/off (via left encoder)
- **EXT PWR:** Toggle external power

## Quick Reference

### Layer Activation

- **BASE:** Default layer
- **LOWER:** Hold left thumb inner key
- **RAISE:** Hold right thumb outer key
- **ADJUST:** Hold both LOWER + RAISE simultaneously

### Common Shortcuts (macOS)

- **CMD + C:** Copy (thumb CMD + C)
- **CMD + V:** Paste (thumb CMD + V)
- **CMD + Tab:** Switch apps (thumb CMD + Tab)
- **CMD + Space:** Spotlight (thumb CMD + Space)

### Bluetooth Pairing

1. Hold RAISE to access BT controls
2. Press BT1-BT5 to select a profile
3. Pair with your device
4. Use BT CLR to clear all connections if needed

### RGB Customization

1. Hold LOWER + RAISE together
2. Use HUE +/- to change colors
3. Use BRI +/- to adjust brightness
4. Twist left encoder to toggle RGB on/off
